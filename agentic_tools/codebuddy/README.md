# CodeBuddy

CodeBuddy is Tencent Cloud's proprietary AI coding assistant CLI, distributed as `@tencent-ai/codebuddy-code` and native CodeBuddy Code archives from https://www.codebuddy.ai. docker-agentic installs the opt-in `codebuddy` binary at `/opt/agentic-tools/codebuddy/bin/codebuddy`; use is governed by Tencent's product/license terms.

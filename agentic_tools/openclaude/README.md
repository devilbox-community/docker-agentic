# OpenClaude

OpenClaude is Gitlawb's community coding-agent CLI for OpenAI-compatible, local, and cloud LLM providers; upstream: https://github.com/Gitlawb/openclaude. It is distributed under MIT modifications with an Anthropic/proprietary Claude Code source notice, so docker-agentic installs it only as opt-in at `/opt/agentic-tools/openclaude/bin/openclaude`.

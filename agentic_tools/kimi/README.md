# Kimi Code

Kimi Code is Moonshot AI's Apache-2.0 terminal AI agent CLI from https://github.com/MoonshotAI/kimi-cli and https://code.kimi.com, installed as the opt-in `kimi` binary at `/opt/agentic-tools/kimi/bin/kimi`. It authenticates through Kimi/Moonshot login or API-key configuration at runtime.

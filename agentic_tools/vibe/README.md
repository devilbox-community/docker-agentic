# Mistral Vibe

Mistral Vibe is Mistral AI's Apache-2.0 open-source coding-agent CLI from https://github.com/mistralai/mistral-vibe, installed as the opt-in `vibe` binary at `/opt/agentic-tools/vibe/bin/vibe` plus `vibe-acp` for ACP integrations. It uses Mistral authentication or `MISTRAL_API_KEY` at runtime.
